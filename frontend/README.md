# Frontend

This folder contains the SPA shell and static assets used by Django.

Run the project:

```powershell
.\.venv\Scripts\python.exe manage.py runserver
```

Open:

```text
http://127.0.0.1:8000
```

The `/config.js` response is generated by Django at runtime, so same-domain deploys work without editing frontend files.

If your backend runs on another host or port, update `.env` and restart Django.
